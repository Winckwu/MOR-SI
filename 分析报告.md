## 第二部分：训练集构建方法与分析

### 2.1 训练集标注方法论

本研究采用**演绎法**构建训练集，核心思路：

```
理论概念 → 操作化定义 → 关键词词典 → 规则标注
```

**具体步骤:**
1. 从Berwick & Christia (2018)论文中提取三种国家能力的理论定义
2. 根据论文引用的测量指标和实证研究，构建关键词词典
3. 在合同名称中匹配关键词，计算各类别得分
4. 选择置信度≥60%的样本作为高质量训练数据

## 第三部分：特征工程分析

### 3.1 文本向量化方法: TF-IDF

| 参数 | 设置 |
|------|------|
| 分词方式 | 字符级分词 + Bigram |
| 最大特征数 | 5,000 |
| N-gram范围 | (1, 2) |

**选择理由:**
1. 中文合同名称通常较短，字符级分词能捕获更细粒度的语义
2. Bigram有助于捕获常见的双字词组合（如"税务"、"建设"）
3. TF-IDF能有效降低常见词（如"采购"、"项目"）的权重

## 第七部分：方法论讨论与局限性

### 7.1 研究方法优势

1. **理论驱动:** 关键词基于学术文献，具有理论效度
2. **可解释性:** 每个分类结果都有明确的关键词依据
3. **高准确率:** SVM模型测试集准确率达99.37%
4. **可扩展性:** 可添加新关键词或调整权重

### 7.2 方法局限性

1. **关键词覆盖不完全:**
   - 约25%样本无法匹配到关键词
   - 需依赖模型泛化能力

2. **多类别重叠问题:**
   - 部分合同可能涉及多种能力
   - 如"国土资源监察"同时涉及汲取和合规能力
   - 当前采用最高得分类别，未做多标签分类

3. **语义理解局限:**
   - 关键词匹配无法捕获深层语义
   - "办公设备采购"的分类取决于采购单位性质
   - 未利用采购人、所属行业等上下文信息

### 7.3 未来改进方向

1. 引入预训练语言模型 (BERT/RoBERTa) 提升语义理解
2. 结合采购人、行业等元数据构建多模态分类器
3. 采用多标签分类处理能力重叠问题
4. 引入主动学习优化标注边界样本
